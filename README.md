# ⚒️ Technically Restarting Design System

> *Crafted with Curiosity.*

A reusable HTML and CSS design system documenting my front-end development journey.

Originally created as a Codecademy challenge project, this repository has evolved into my personal design system—a collection of reusable colors, typography, components, and design principles that will continue to grow alongside my skills.

---

## ✨ Features

- 🎨 Reusable color palette
- 🔤 Typography showcase using Google Fonts
- 📝 Text style reference
- 🧩 Reusable UI components
- 🎯 CSS custom properties (variables)
- 📐 Responsive-friendly layout
- 💡 Clean semantic HTML structure

---

## 🎨 Design Philosophy

The visual identity is inspired by three ideas:

- ⚒️ Traditional craftsmanship
- 🌿 Continuous growth and curiosity
- 💻 Modern technology

The goal is to blend timeless design with clean, maintainable code.
